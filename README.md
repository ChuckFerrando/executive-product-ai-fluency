# Executive Product AI Fluency

## Purpose
This repository defines the AI models, tools, and product development patterns that executive product leaders must understand, evaluate, and apply.

It is designed as a decision-support system for navigating the rapidly evolving AI landscape.

---

## What This Repo Is

- A curated system of AI models, tools, and capabilities
- A guide for applying AI in product contexts
- A decision framework for evaluating AI solutions
- A reference for modern product development patterns

---

## What This Repo Is NOT

- A complete catalog of AI tools
- A beginner tutorial on machine learning
- A collection of unstructured notes
- A purely technical deep-dive repository

---

## Who This Is For

- Executive product leaders
- Product operators working with AI-enabled products
- Builders integrating AI into product workflows

---

## How to Use This Repo

- Start with `START_HERE.md`
- Review `REPO_MAP.md` for structure
- Explore AI categories, tools, and use cases
- Use evaluation frameworks to guide decisions
- Reference implementation patterns for real-world application
